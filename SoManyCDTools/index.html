Automated, reliable tests

Green-field approach:

Always test your software in a fresh environment.

  • Create the runtime environment from scratch.
  • Install or compile your software
  • start it
  • run the tests
  • Throw the entire environment away.

Virtual machines

Test on multiple runtime environments

  • Different OS
  • Different JavaEE container
  • Different Java version

Vagrant

Easy setup of the runtime environment.

Based on a script.

Put the script under version control.

Huge repository of predefined scripts: VagrantBox.es

Update your OS in five minutes

vagrant halt
vagrant destroy

Edit the config file.

vagrant up

Ansible and Puppet

Install software in a Vagrant box.

Based on a script.

Reinstall your installation in a minute:

vagrant halt vagrant destroy

Edit the config file.

vagrant up

Docker

Virtual machine for Linux

Similar to Vagrant - but needs less resources

Usually used in production

Orcas
